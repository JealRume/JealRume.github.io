# 🚀 Personal Portfolio | Jesús Ruiz

Hi! I'm Jesús Ruiz, a **Riwi Developer** passionate about creating beautiful and functional websites. This repository contains the source code for my personal portfolio, designed with a modern "Cyberpunk" aesthetic and focused on user experience.

---

## 🛠️ Technologies Used

For this project, I used a set of technologies that allow me to create fast and attractive interfaces:

* **Frontend:** Semantic HTML5, CSS3 (Custom Properties, Flexbox & Grid).

* **Design:** Dark Mode aesthetic with Neon effects.

* **Versioning:** Git & GitHub.

* **Main Stack:** 🐍 Python, ☕ Java, HTML, CSS

---

## 🌟 Features

- **Responsive Design:** Optimized for mobile and desktop devices.

- **Pets Section:** A space dedicated to my companions (Jacob, Shelby, and Orochi).

- **Intuitive Navigation:** Persistent sidebar for quick access to all sections.

- **CV Download Button:** Direct integration for recruiters.
